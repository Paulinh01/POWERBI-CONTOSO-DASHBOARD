## 📌 Sobre o Projeto

Este projeto consiste no desenvolvimento de um Dashboard Executivo utilizando a base **Contoso Retail**, com foco na análise de desempenho comercial por meio de consultas SQL e visualizações interativas no Power BI.

O objetivo é transformar dados brutos em informações estratégicas para apoiar a tomada de decisão em diferentes níveis da organização.

---

# 🎯 Objetivos

- Monitorar Receita, Custos e Lucro
- Avaliar a performance das lojas
- Identificar produtos mais rentáveis
- Comparar fabricantes, categorias e continentes
- Construir um dashboard executivo para tomada de decisão
